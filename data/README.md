# Dataset

This project uses the public Concrete dataset for binary image classification.

**Download link:** https://data.mendeley.com/datasets/5y9wdsg2zt/2

**Dataset details:**
- Total samples: 40000 images
- Classes: Positive, Negative (binary classification)
- Split: 70% train, 15% validation, 15% test

Due to size constraints, the dataset is not included in this repository.
Please download from the link above and place in this directory.
