# Dataset

This project uses the public Concrete dataset for binary image classification.

**Download link:** https://data.mendeley.com/datasets/5y9wdsg2zt/2

**Dataset details:**
- Total samples: 40,000 images (20,000 per class)
- Classes: Positive (defect present), Negative (no defect)
- Image size: 227Ã—227 pixels, RGB
- Split: 70% train, 15% validation, 15% test

**Setup:**
1. Download the dataset from the link above
2. Extract images into:
   - `data/Negative/` (20,000 images)
   - `data/Positive/` (20,000 images)

Due to size constraints, the dataset is not included in this repository.
